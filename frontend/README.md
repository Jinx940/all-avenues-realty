# Frontend

React + Vite client for the All Avenues Realty workspace.

## Commands

- `npm run dev`
- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Environment

- `VITE_API_URL` is optional.
- If it is empty, the frontend uses the Vite proxy for `/api`.
- If it is set, requests go directly to that backend origin with credentialed cookies.
